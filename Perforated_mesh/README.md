### Description 
This file shows how to create perforated metal material with Honeybee.
In order to use this file you need to copy [perforate.cal](https://raw.githubusercontent.com/mostaphaRoudsari/hydra/master/Perforated_mesh/perforate.cal) to C:\radiance\lib

This file has been submitted by [mostaphaRoudsari](https://github.com/mostaphaRoudsari)
